Comandos del Panel de Control en Linux

Explora los comandos más útiles y aprende para qué sirven en el sistema Linux.

1. ls

Descripción: Lista los archivos y directorios en el directorio actual.

Uso común: ls -l muestra una lista detallada de archivos, incluyendo permisos y fechas de modificación.

2. cd

Descripción: Cambia el directorio de trabajo actual.

Uso común: cd /home/usuario para cambiar al directorio de usuario.

3. pwd

Descripción: Muestra la ruta completa del directorio actual.

Uso común: pwd muestra la ubicación exacta en la que te encuentras.

4. mkdir

Descripción: Crea un nuevo directorio.

Uso común: mkdir nuevo_directorio crea un directorio con el nombre "nuevo_directorio".

5. rmdir

Descripción: Elimina un directorio vacío.

Uso común: rmdir directorio_vacio elimina un directorio vacío.

6. touch

Descripción: Crea un archivo vacío o actualiza la fecha de acceso de un archivo.

Uso común: touch archivo.txt crea el archivo "archivo.txt" si no existe.

7. cp

Descripción: Copia archivos o directorios.

Uso común: cp archivo1.txt archivo2.txt copia "archivo1.txt" a "archivo2.txt".

8. mv

Descripción: Mueve o renombra archivos y directorios.

Uso común: mv archivo.txt /nuevo/directorio/ mueve el archivo al nuevo directorio.

9. rm

Descripción: Elimina archivos o directorios.

Uso común: rm archivo.txt elimina el archivo especificado.

10. chmod

Descripción: Cambia los permisos de archivo.

Uso común: chmod 755 archivo.sh cambia los permisos de "archivo.sh".

11. chown

Descripción: Cambia el propietario y el grupo de un archivo.

Uso común: chown usuario:grupo archivo.txt cambia el propietario de "archivo.txt".

12. cat

Descripción: Muestra el contenido de un archivo.

Uso común: cat archivo.txt muestra el contenido de "archivo.txt".

13. echo

Descripción: Imprime texto o variables en la terminal.

Uso común: echo "Hola Mundo" imprime "Hola Mundo" en la terminal.

14. top

Descripción: Muestra los procesos en ejecución y el uso de recursos.

Uso común: top muestra una lista de procesos con el uso de CPU y memoria.

15. ps

Descripción: Muestra información sobre los procesos en ejecución.

Uso común: ps aux muestra todos los procesos activos.

16. kill

Descripción: Mata un proceso en ejecución.

Uso común: kill 1234 mata el proceso con el PID 1234.

17. df

Descripción: Muestra el espacio libre en los sistemas de archivos.

Uso común: df -h muestra el espacio en disco en un formato legible.

18. du

Descripción: Muestra el uso de espacio de archivos y directorios.

Uso común: du -sh muestra el tamaño total del directorio actual.

19. grep

Descripción: Busca patrones dentro de archivos.

Uso común: grep 'patrón' archivo.txt busca "patrón" dentro del archivo.

20. man

Descripción: Muestra el manual de un comando.

Uso común: man ls muestra el manual del comando "ls".
