Estructura principal de directorios en Linux

La estructura de carpetas en Linux sigue una jerarquía tipo árbol, donde todo parte del directorio raíz /. Esta organización está definida por el estándar FHS (Filesystem Hierarchy Standard), que establece qué contiene cada carpeta principal del sistema.

Carpeta Descripción
/Directorio raíz. Todo el sistema parte de aquí.
/binBinarios esenciales del sistema (comandos como ls, cp, mv, etc.)
/bootArchivos necesarios para el arranque del sistema (kernel, GRUB, etc.)
/devArchivos de dispositivos (discos, terminales, etc.)
/etcArchivos de configuración del sistema
/homeDirectorios personales de los usuarios
/libBibliotecas compartidas necesarias para ejecutar los binarios en /bin
/mediaPunto de montaje para dispositivos externos (USB, CD, etc.)
/mntPunto de montaje temporal para sistemas de archivos
/optSoftware adicional instalado manualmente
/procInformación del sistema y procesos en tiempo real (virtual)
/rootCarpeta personal del usuario root
/runInformación de estado del sistema desde el arranque
/sbinBinarios esenciales para administración del sistema
/srvDatos para servicios ofrecidos por el sistema (servidores web)
/tmpArchivos temporales
/usrProgramas y archivos de usuario compartidos
/varArchivos variables como logs, colas de impresión, bases de datos
