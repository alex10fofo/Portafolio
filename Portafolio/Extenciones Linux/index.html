Extensiones en Linux

Personaliza y potencia tu experiencia en Linux con extensiones útiles.

¿Qué son las extensiones en Linux?

Las extensiones en Linux permiten ampliar las funcionalidades del entorno de escritorio, como GNOME, KDE o XFCE. Estas herramientas pueden cambiar la apariencia, agregar accesos rápidos, gestionar ventanas de manera más eficiente y mucho más.

Extensiones populares para GNOME

  • Dash to Dock: Convierte el dock en una barra tipo macOS.
  • GSConnect: Permite integrar tu teléfono Android con el escritorio.
  • Clipboard Indicator: Muestra el historial del portapapeles.
  • Sound Input & Output Device Chooser: Cambia fácilmente dispositivos de audio.

¿Cómo instalar extensiones en GNOME?

Desde el navegador

1. Visita extensions.gnome.org

2. Instala el complemento de navegador "GNOME Shell integration".

3. Habilita las extensiones directamente desde el sitio.

Por consola

sudo apt install gnome-shell-extensions
gnome-extensions enable nombre.extension

Extensiones en KDE

En KDE se utilizan los "plasmoides" y scripts para agregar funcionalidades.

  • Latte Dock: Dock animado con muchas opciones de personalización.
  • KRunner Plugins: Extiende las búsquedas inteligentes.
  • KDE Connect: Comunicación entre PC y Android.

Instalación

Desde Discover (tienda de aplicaciones) o mediante terminal:

plasmapkg2 --install archivo.plasmoid

O desde store.kde.org

Extensiones para terminal (CLI)

  • fzf: Fuzzy finder para buscar archivos rápidamente.
  • exa: Reemplazo moderno para ls.
  • bat: Vista mejorada para archivos de texto con resaltado.

Instalación

sudo apt install fzf exa bat
# o usando Homebrew en Linux
brew install fzf exa bat

Recomendaciones adicionales

Consulta estas fuentes para más extensiones:

  • extensions.gnome.org
  • store.kde.org
  • GitHub (repositorios de extensiones)
