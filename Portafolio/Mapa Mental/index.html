Línea del Tiempo de Sistemas Operativos

Monotarea

Solo ejecuta una tarea a la vez. Ejemplo: MS-DOS.

Monousuario

Diseñado para un solo usuario a la vez. Ejemplo: Windows 95.

UniProceso

Diseñado para un solo proceso a la vez. Ejemplo: Windows 95.

Multiproceso

Permite mantener varios procesos en ejecución simultáneamente.

Multitarea

Ejecuta múltiples tareas al mismo tiempo. Ejemplo: UNIX moderno.

Multiusuario

Varios usuarios pueden usar el sistema al mismo tiempo. Ejemplo: Linux.
